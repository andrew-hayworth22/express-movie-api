Express Movie REST API

  • GET localhost:5000/api/movies
    • Get list of all movies
  • GET localhost:5000/api/movies/{id}
    • Get one movie by its ID
  • POST localhost:5000/api/movies
    • Create a movie
    • Accepts this JSON request:
      {
         "name": "The Godfather Part II"
         "releaseDate": "1974-12-20"
      }
